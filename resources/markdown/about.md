# 未来図倉庫について
｢未来図倉庫｣は、ポップカルチャースタジオが運営しているフリー素材集です。
イラストやBGM、Live2Dなど幅広いジャンルの素材を日々制作しています。
様々なニーズに対応できるよう、更新していきます。
お役立てください。

## 素材
ゲーム制作やウェブサイト、印刷物、企画書など、利用規約の範囲内であれば商用･非商用を問わず無料でご利用いただけます。
素材データはそジャンルによって、れぞれの形式が異なります。ご了承ください。
